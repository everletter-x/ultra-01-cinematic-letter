# Ultra Premium 01 - Cinematic Letter

A luxurious, movie-like letter experience built with Next.js 13, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- Dark luxury theme with gold accents
- Chapter-based vertical scrolling
- Cinematic transitions and animations
- Ken Burns photo effects
- Floating music controls
- Responsive design
- Fully configurable via `public/config.json`

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the letter.

## Configuration

Edit `public/config.json` to customize:
- Recipient and sender names
- Title and message
- Photo paths
- Chapter content
- Music settings
- Captions and closing text

## Build

```bash
pnpm build
pnpm start
```

## Tech Stack

- Next.js 13
- TypeScript
- Tailwind CSS
- Framer Motion
